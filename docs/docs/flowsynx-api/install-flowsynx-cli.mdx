# Install the FlowSynx CLI

Install the FlowSynx CLI as the main tool for running FlowSynx-related tasks

## Step 1: Install the FlowSynx CLI
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs groupId="operating-systems">
  <TabItem value="linux" label="Linux">
    ### Install from Terminal
    Install the latest Linux FlowSynx CLI to /usr/local/bin:
    ```
    wget -q https://raw.githubusercontent.com/FlowSynx/Cli/master/install/install.sh -O - | /bin/bash
    ```

    ### Installing a specific CLI version
    The following example shows how to install FlowSynx CLI version 0.1.0.
    ```
    wget -q https://raw.githubusercontent.com/FlowSynx/Cli/master/install/install.sh -O - | /bin/bash -s 0.1.0
    ```

    ### Install without sudo
    If you do not have access to the sudo command or your username is not in the sudoers file, you can install FlowSynx to an alternate directory via the FLOWSYNX_INSTALL_DIR environment variable. This directory must already exist and be accessible by the current user.
    ```
    wget -q https://raw.githubusercontent.com/FlowSynx/Cli/master/install/install.sh -O - | FLOWSYNX_INSTALL_DIR="$HOME/flowsynx" /bin/bash
    ```

    ### Installing a specific FlowSynx version without sudo
    The following example shows how to install FlowSynx CLI version 0.1.0.
    ```
    wget -q https://raw.githubusercontent.com/FlowSynx/Cli/master/install/install.sh -O - | FLOWSYNX_INSTALL_DIR="$HOME/flowsynx" /bin/bash -s 0.1.0
    ```

## Step 2: Verify the installation
Verify the CLI is installed by restarting your terminal/command prompt and running the following:
```
synx -h
```
