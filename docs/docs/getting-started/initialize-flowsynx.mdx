# Initialize FlowSynx

Fetch the FlowSynx binaries and install them locally using `synx init`.  
Now that youâ€™ve installed the FlowSynx CLI, use the CLI to initialize FlowSynx on your local machine.

## Step 1: Open terminal

## Step 2: Run the init CLI command
Install the latest FlowSynx binaries:
```
synx init
```
After executing this command, FlowSynx starts to be initialized.

## Step 3: Verify FlowSynx version
```
synx version
```

#### Output:
CLI version: 0.1.0  
FlowSynx version: 0.1.0
