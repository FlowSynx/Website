## Step 2: Run the FlowSynx
```
flowsynx run
```
After executing this command, FlowSynx starts to be running and the logs like below will be displayed.


## Step 3: Calling API
Open a new terminal and enter the below command (in this example **`List`** command will be called to getting list from given path):
<Tabs groupId="httpApi">
  <TabItem value="bashHttpApi" label="HTTP API (Bash)">
  ```
  curl -X POST -H "Content-Type: application/json" -d '{ "path": "/", "full": true, "caseSensitive": false, "recurse": false, "sorting": "kind asc" }' http://localhost:5860/storage/list
  ```
  </TabItem>
  <TabItem value="powershellHttpApi" label="HTTP API (PowerShell)">
  ```
  Invoke-RestMethod -Method Post -ContentType 'application/json' -Body '{ "path": "C:/", "full": true, "caseSensitive": false, "recurse": false, "sorting": "kind asc" }' -Uri 'http://localhost:5860/storage/list'
  ```
  </TabItem>
</Tabs>

After executing the command, the result will be returned to you in the form of `json` data.
